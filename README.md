## Getting Started
First, clone the git

```bash
git clone https://github.com/Jinyoung-s/fulhaus-fe.git

cd fulhaus-fe
```

Second, install the libraries.

```bash
npm install
```

Third, run the server
 ```bash
 npm run start
 ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
